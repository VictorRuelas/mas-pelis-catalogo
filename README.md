# mas-pelis-catalogo
Plataforma frontend de catálogo de películas (streaming/alquiler) desarrollada con HTML, CSS (BEM) y JavaScript modular (ESM). Incluye funcionalidades de filtrado por género, búsqueda de texto y gestión del carrito de compras/alquiler utilizando la estructura de datos Map, optimizado para ser un sitio estático de alto rendimiento.
